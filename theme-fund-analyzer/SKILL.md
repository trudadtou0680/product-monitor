# Theme Fund Analyzer

## 定位

围绕用户给定的主题产品池，完成基金公开数据抓取、口径校验、收益/回撤计算、规模合并和名单内排名。输出必须面向内部业务人员，先给排序结果和关键结论，再给口径、来源、冲突和缺口。

本 skill 只处理公开基金数据分析，不承诺收益、不替代投顾建议、不编造缺失数据。

## 使用参考文件

按任务读取对应参考文件：

- 数据源、字段来源、接口降级：读取 `references/data-sources.md`。
- 时间区间、排序、排名、多份额、异常处理：读取 `references/analysis-rules.md`。
- 主题产品池事实源：读取并维护 `references/product-pools.md`。

## 推荐工作流

1. 明确用户输入：主题范围、时间区间、排序字段、提取数量、是否导出文件。
2. 若用户提到的主题已存在于 `references/product-pools.md`，直接使用本地产品池。
3. 若用户提到的是全新主题，先通过东方财富概念板块接口匹配细分概念板块；匹配成功后优先使用东方财富板块主题基金接口生成临时基金池，接口无返回时再使用概念板块名称和用户主题关键词从基金目录生成临时基金池，并在输出中标注概念板块代码。若无法匹配概念板块或无法形成基金池，提示用户提供产品池。
4. 若用户直接提供新的产品池，先用脚本写入 `references/product-pools.md`：同名主题直接覆盖，新主题新增 `## 主题名` 和产品表。

```bash
python3 scripts/fetch_theme_funds.py --theme 机器人 --set-pool - --update-only <<'EOF'
| 基金代码 | 基金名称 |
| --- | --- |
| 012345 | 示例基金 A |
EOF
```

5. 运行脚本抓取和计算：

```bash
python3 scripts/fetch_theme_funds.py --theme CPO --period 1m --sort return --top 30
```

6. 若用户要求所有主题，使用 `--theme all`；若要求自定义日期，使用 `--period custom --start YYYY-MM-DD --end YYYY-MM-DD`。
7. 用户无特殊要求时，默认表格列为：排名、基金代码、基金名称、合并规模(亿元)、近1日(标注 MM/DD)/排名、近1周/排名、近1月/排名。
8. 对名称代码不一致、代码无法匹配、多候选、净值不足、规模缺失、接口失败，必须单独列出，不得静默忽略。

## 脚本能力

`scripts/fetch_theme_funds.py` 支持：

- 从 `references/product-pools.md` 读取主题产品池。
- 使用 `--set-pool <文件路径或-> --theme <主题名>` 接收用户给出的产品清单，自动覆盖或新增 `references/product-pools.md` 中对应主题；只维护产品池时加 `--update-only`。
- 本地不存在主题时，可通过东方财富概念板块接口匹配 CPO、PCB、先进封装等细分概念板块，并优先使用东方财富板块主题基金接口生成临时产品池；接口无返回时，再基于概念板块名称和用户主题关键词从基金目录生成临时产品池。
- 以产品池参考文件中的基金代码为事实源；基金名称用于展示、辅助校验和代码缺失时的降级匹配。
- 对多份额产品合并最近报告期规模。
- 同一产品多份额只展示一行，不得让 A/C/E 等份额重复占用排名；存在 A 类份额时，收益率和最大回撤使用 A 类份额口径，规模仍合并所有份额。
- 支持 `latest-day`、`1w`、`1m`、`3m`、`6m`、`1y`、`ytd`、`custom` 区间。
- 支持按收益率、最大回撤、合并规模排序，并输出 Markdown、CSV 或 JSON；使用 `--detailed` 可输出单一区间收益率和最大回撤明细。

## 输出原则

- 先给名单内排序结果，再给冲突/缺口。
- 用户无特殊要求时，默认表格列固定为 `排名`、`基金代码`、`基金名称`、`合并规模(亿元)`、`近1日(MM/DD)/排名`、`近1周/排名`、`近1月/排名`。
- 近1日必须在列头标注实际净值日期，格式为 `MM/DD`。
- 收益率列必须带各自指标的名单内排名，格式为 `数值 / 排名`。
- 用户明确要求回撤时，回撤列必须带回撤指标排名，格式为 `数值 / 排名`。
- 排名只在同一主题池内计算，不做跨主题混排，除非用户明确要求。
- 收益率越大排名越靠前；回撤越小排名越靠前，最大回撤用负数展示时数值越接近 0 越靠前。
- 所有数值必须说明单位；收益率和回撤使用百分比，规模使用亿元。
- 公开接口失效或字段缺失时，输出“无法获取/待确认”，不得推测。
- 投资相关表达必须保留风险提示和数据时点，不得承诺未来收益。
